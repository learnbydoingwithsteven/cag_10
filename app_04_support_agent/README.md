# App 04: Customer Support Agent

**Conversational CAG with Memory**

This application demonstrates how to maintain conversation history and context in a support setting.

## Features
- **Context Retention**: Remembers previous turns in the conversation.
- **Knowledge Retrieval**: Fetches relevant support docs based on user queries.
- **Tone Adaptation**: Responds in a helpful, professional customer support persona.
