# App 09: Contract Compliance Checker

**Rule-based CAG**

This application verifies contracts against a set of predefined rules (e.g., GDPR, Payment Terms).

## Features
- **Clause Extraction**: Identifies specific clauses in legal documents.
- **Rule Validation**: Checks extracted clauses against a rulebook.
- **Risk Flagging**: Highlights non-compliant sections.
