# App 05: Financial Report Analyzer

**Structured Data CAG**

This application extracts insights from structured financial reports (PDFs, Tables) using CAG.

## Features
- **Table Parsing**: Understands rows/columns in financial data.
- **Ratio Analysis**: Computes key financial ratios on the fly.
- **Trend Detection**: Compares data across different years/quarters.
